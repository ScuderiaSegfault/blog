We provide basic apps and tools to use our BLE and USB devices from the Web.
Please note that all these apps required the use of a Chromium-based browser with _Experimental Web Platform Features_ (enable this feature:
`chrome://flags/#enable-experimental-web-platform-features`).

Currently we provide the following apps:

* [**Reference Receiver Config**]({{ "/apps/ir_nec/" | relative_url }}): Configure the IR NEC reference receiver and verify its operation.
