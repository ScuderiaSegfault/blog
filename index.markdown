Hi, we are [Scuderia Segfault](https://scuderia-segfault.at/) the [Roboracer](https://roboracer.ai/) team at [TU Wien](https://tuwien.ac.at/).
In this blog we want to write about the work we are currently working on (and willing to publish).
We are making no guarantees about publishing frequency (not unlike FastDDS...) or that we will publish everything, but we will try to publish as much as we possibly can, while still staying competitive.

Our philosophy for this blog:

1. Publish things that benefit everyone as a blog post and as code
2. Publish things that benefit other teams as a blog post
3. Keep things to ourselves that are embarassing, not yet finished, or haven't been used in a race yet
